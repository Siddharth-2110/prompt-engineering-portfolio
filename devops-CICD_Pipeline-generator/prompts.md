# DevOps CI/CD Pipeline Generator Prompts

## 🎯 Objective

Generate structured, step-by-step DevOps and CI/CD documentation.

---

## 🔹 Prompt 1: CI/CD Pipeline Setup

**Prompt:**
Create step-by-step documentation for setting up a CI/CD pipeline using Jenkins, Docker, and AWS EC2.

**Output:**

1. Launch an EC2 instance
2. Install Docker and Jenkins
3. Configure Jenkins pipeline
4. Connect GitHub repository
5. Build Docker image
6. Deploy container on EC2

**Techniques Used:**

* Step-by-step structuring
* Logical sequencing
* Instruction clarity

---

## 🔹 Prompt 2: Docker Deployment Guide

**Prompt:**
Write a beginner-friendly guide to deploy a Dockerized application.

**Output:**

1. Install Docker
2. Pull or build Docker image
3. Run container using docker run
4. Expose required ports
5. Verify application

**Techniques Used:**

* Simplified technical steps
* Sequential flow
* Beginner-friendly language

---

## 🔹 Prompt 3: Multi-App Deployment

**Prompt:**
Explain how to deploy multiple applications on a single EC2 instance using Docker Compose.

**Output:**
Use Docker Compose to define multiple services in a single file. Assign different ports to each application and run them together using one command.

**Techniques Used:**

* Concept simplification
* Structured explanation
* Practical use case

---

## 🧠 Key Learnings

* Structured prompts produce clean documentation
* Step-by-step format improves usability
* Clear instructions reduce ambiguity
