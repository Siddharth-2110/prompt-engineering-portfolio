## 📂 Portfolio Projects

### 1. 📧 Email Generator (Professional Communication)

**Goal:** Generate clean, ready-to-use emails with tone control

**Prompt:**
Act as a corporate professional. Write a semi-formal follow-up email for a job opportunity. Keep it polite, concise, and under 120 words.

**Output:**
Hi [Name],
I hope you're doing well. I wanted to follow up regarding the opportunity discussed earlier. I would be grateful if you could keep me in mind for any suitable roles. I am open to learning opportunities and would appreciate guidance within a positive work environment.
Thank you for your time and consideration.

**Techniques Used:**

* Role-based prompting
* Length constraint
* Tone control

### 3. 🔁 Response Refinement System

### 4. 🧩 Technical Simplifier

**Goal:** Explain complex topics simply

**Prompt:**
Explain Docker to a beginner using a simple real-life analogy in under 100 words.

**Output:**
Docker is like a lunchbox. Instead of carrying raw ingredients and cooking everywhere, you pack your entire meal ready-to-eat. Similarly, Docker packages an application with everything it needs, so it runs anywhere without setup issues.

**Techniques Used:**

* Audience targeting
* Analogy-based explanation
* Constraint-based clarity
